## MVP ソリューション詳細

### SL001: 3x3ビンゴボード作成

**説明:**
年間目標を3x3（9マス）のビンゴ形式で入力・表示する、コア機能。

**対応課題:**
- PR001: 目標を忘れる → ビンゴで一望
- PR002: 複雑すぎる → 9マスでシンプル

**機能:**
- 9マスのグリッド表示
- 各マスへの目標テキスト入力（タップで編集）
- 達成/未達成の切り替え（タップでトグル）
- 年度選択（2025, 2026, 2027, ...）

**受け入れ基準:**
- [ ] 3x3のグリッドが画面中央に表示される
- [ ] 各マスをタップすると目標を入力できる
- [ ] 達成マスをタップすると達成済みに変わる
- [ ] 年度を切り替えると別のボードが表示される

---

### SL002: ビンゴ達成演出

**説明:**
縦・横・斜めが揃った時にアニメーションで祝福し、達成感を演出。

**対応課題:**
- PR003: 達成感を感じられない → 演出で解決

**機能:**
- ビンゴ判定ロジック（縦3列 + 横3列 + 斜め2列 = 最大8ビンゴ）
- 達成時のアニメーション（紙吹雪、光る効果など）
- 祝福メッセージ（「BINGO!」「おめでとう！」）
- 達成ライン数のカウント（「2ビンゴ達成！」）

**受け入れ基準:**
- [ ] 縦/横/斜めが揃うとビンゴと判定される
- [ ] ビンゴ時にアニメーションが再生される
- [ ] 複数ライン同時達成にも対応
- [ ] 達成ライン数が表示される

---

### SL003: ポジティブUI設計

**説明:**
達成マスを強調し、未達成マスは控えめに表示することで、ネガティブな印象を軽減。

**対応課題:**
- PR004: できなかったことが辛い → ポジティブUIで解決

**機能:**
- 達成マスの色付け/アイコン（チェックマーク等）
- 未達成マスはグレーアウト or 薄い表示
- 「あと○マスでビンゴ」のポジティブな表現
- 進捗率の表示（「5/9 達成」）

**受け入れ基準:**
- [ ] 達成マスは目立つ色で表示される
- [ ] 未達成マスは控えめに表示される
- [ ] 進捗が数字で表示される
- [ ] ネガティブな表現がUIに含まれない

---

### SL004: レスポンシブWebアプリ

**説明:**
PC/スマホ両対応のWebアプリとして開発し、どちらでも快適に使える。

**対応課題:**
- PR006: PCで使えない → Web版で解決

**機能:**
- モバイルファーストのUI（375px〜）
- タブレット/PC対応（768px〜、1024px〜）
- PWA対応（ホーム画面に追加可能）

**受け入れ基準:**
- [ ] スマホで表示が崩れない（375px幅）
- [ ] PCで表示が崩れない（1024px幅）
- [ ] 操作がスムーズ（タップ/クリック両対応）
