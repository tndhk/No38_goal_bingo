# プロダクト定義: bingo_goal_app

## プログラム定義

### program_name
年度別目標ビンゴ（bingo_goal_app）

### theme_strategy
「シンプルに続けられる年間目標管理」をテーマに、ビンゴ形式のゲーミフィケーションで92%の挫折者を救う

### Mission
年間目標を「立てて終わり」にさせない。シンプルなビンゴ形式で、誰もが目標達成の喜びを体験できるようにする。

### Vision
「目標を忘れない」「達成感を得られる」「続けられる」の3つを兼ね備えた、最もシンプルな年間目標管理アプリになる。

### Value
| 価値 | 説明 |
|------|------|
| シンプル | 3x3の9マスから始められる敷居の低さ |
| 視覚的達成感 | ビンゴが揃う瞬間の達成感演出 |
| 年度軸 | 年間目標を一望できる唯一のビンゴアプリ |
| 継続性 | 完璧主義を緩和するポジティブUI |

### product_vision
For: 新年の抱負で毎年挫折する人、目標管理アプリが複雑すぎると感じる人
Who: 年間目標を立てるが、5月には忘れてしまう
The: 年度別目標ビンゴ
Is a: 目標管理Webアプリ
That: 3x3ビンゴ形式で目標を視覚化し、達成時の演出で継続モチベーションを維持
Unlike: 9x9マンダラチャートアプリ（複雑）、習慣化アプリ（日次管理のみ）
Our product: 年度軸での目標管理と、ビンゴ達成の楽しさを両立

### organization
- 開発形態: 個人開発
- プラットフォーム: Web（レスポンシブ、モバイルファースト）
- 技術スタック: 検討中（Sveltekit or Next.js想定）

## ポジションステートメント

### when_context（いつ・どんな状況で）
新年や年度始めに「今年こそは」と目標を立てる時。
または、既存の目標管理アプリが複雑すぎて挫折した時。

### what_unique（何がユニークか）
- 「年度別」×「ビンゴ形式」の唯一の組み合わせ
- 3x3（9マス）から始められるシンプルさ
- 縦・横・斜めが揃った時の達成感演出

### how_features（どんな機能で実現するか）
| 機能 | 説明 |
|------|------|
| 3x3/5x5ビンゴボード | 年間目標を9または25マスに配置 |
| ビンゴ達成演出 | 列が揃った時のアニメーション |
| 年度切り替え | 2025年、2026年など年度別に管理 |
| 進捗の可視化 | 達成済みマスのハイライト |

### where_origin（どこから生まれたか）
- 個人の課題: 毎年、新年の抱負を忘れてしまう
- 市場調査: 年度別ビンゴ形式の競合が不在
- ユーザー調査: 92%が挫折、42%が目標を忘れる

### why_need（なぜ必要か）
| 課題 | 本プロダクトの解決策 |
|------|---------------------|
| 目標を忘れる（42%） | ビンゴボードで常に一望 |
| 達成感がない | ビンゴ達成時の演出 |
| 複雑すぎる | 3x3から開始可能 |
| 年間で意識薄れる | 年度軸で12ヶ月を管理 |

### who_target（誰のためか）

**プライマリターゲット:**
- 20-40代の社会人
- 新年の抱負を毎年立てるが、毎年挫折する人
- 「シンプルなアプリがいい」と感じる人

**セカンダリターゲット:**
- 9x9マンダラチャートが複雑すぎると感じる人
- 手帳やNotionで目標管理していたが続かない人

## 想定ユーザーストーリー

### ストーリー1: 新年の抱負
> 1月1日、太郎さんは「今年こそ」と思い、bingo_goal_appを開く。
> 3x3のビンゴボードに「本を12冊読む」「5kg痩せる」など9つの目標を入力。
> 入力はわずか3分。ビンゴボードを眺めて「今年は揃えるぞ」とワクワクする。

### ストーリー2: 初のビンゴ達成
> 5月、太郎さんは3つ目の目標を達成。縦一列が揃った瞬間、
> 「BINGO!」のアニメーションが表示。嬉しくてスクリーンショットをSNSに投稿。
> 「残り2列も揃えよう」とモチベーションが上がる。

### ストーリー3: 年末の振り返り
> 12月31日、太郎さんは2026年のビンゴを眺める。
> 9マス中6マスが達成済み。ビンゴは2列揃った。
> 「去年は3マスだったから成長した」と実感し、2027年のボードを作成開始。

## 競合との差別化サマリー

| 軸 | 競合 | bingo_goal_app |
|----|------|----------------|
| 形式 | 9x9マンダラが主流 | 3x3/5x5ビンゴ |

| プラットフォーム | モバイルのみが多い | Web（PC/スマホ両対応） |
| 複雑さ | 81マス | 9マスから開始可能 |
