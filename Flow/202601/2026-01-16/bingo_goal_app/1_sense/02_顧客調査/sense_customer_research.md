# 顧客調査レポート: bingo_goal_app

## 調査概要

- 調査日: 2026-01-16
- プロジェクト: bingo_goal_app
- ターゲット: 個人で目標管理をしたい人、新年の抱負を立てる人
- 調査トピック: 目標管理の悩み、既存ツールへの不満、達成率の実態

---

## 1. ターゲットユーザーの実態

### 1.1 新年の抱負に関する統計

| 指標 | 数値 | 出典 |
|------|------|------|
| 新年の抱負を達成できた人 | 8-20% | bondavi調査、スクラントン大学研究 |
| 途中で挫折する人 | 92% | 米国調査 |
| 抱負を忘れてしまう人 | 42% | bondavi調査 |
| 100%以上達成した人 | 6.7% | マンダラチャート協会調査 |

### 1.2 目標への意識の推移

松井証券の2023年調査によると：
- 4月: 78%が目標を意識
- 11月: 54%まで低下
- 5月以降に急激に意識が薄れる傾向

### 1.3 「三日坊主の日」

- アメリカでは1月の第2金曜日を「Quitters Day」と呼ぶ
- 約半数が1月末には新年の抱負を忘れる

## 2. 目標達成を妨げる要因

### 2.1 目標設定の問題

| 問題パターン | 説明 |
|--------------|------|
| 目標が曖昧 | 「もっと運動する」「痩せる」など具体性がない |
| 目標が高すぎる | 新年は「何でもできる」気分で無理な目標を設定 |
| 進捗が分かりにくい | 漠然とした目標は達成度が測れない |
| 目標を思い出さない | 日常で目標に触れる機会がない |

### 2.2 挫折する4つのタイプ

| タイプ | 特徴 |
|--------|------|
| 勢いタイプ | 最初の勢いはあるが計画性がなく途中で挫折 |
| 壮大タイプ | 大きすぎる目標で現実とのギャップに苦しむ |
| 余裕なしタイプ | 日常に追われて目標に取り組む時間がない |
| 遭難タイプ | 目標は立てるが何をしていいか分からない |

### 2.3 心理的要因

- 「意志の弱さ」が原因と思われがちだが、実際は「目標の立て方」に問題
- 1日できないと「失敗した」と感じてしまう完璧主義
- モチベーションに頼りすぎる（仕組み化されていない）

---

## 3. 既存ツールへの不満

### 3.1 UIデザインの問題

- 「UIが洗練されていない」
- 「シンプルでない」
- 「一覧表示がない」
- 「直感的に操作できない」

### 3.2 広告問題

- 無料版は広告が多すぎて使いにくい
- 記録するたびに動画広告が表示される
- 広告非表示のために月額課金が必要

### 3.3 機能面の不満

- 操作が複雑で入力項目が多い
- 振り返り機能が少ない（無料版）
- 日本語対応していないアプリがある
- 登録できる習慣数に制限がある
- リマインド機能がない（一部アプリ）

### 3.4 ソーシャル機能への抵抗

- 「知らない人と写真を投稿し合うのが嫌」
- チームメンバーに依存するストレス

### 3.5 不具合

- 名前変更でアプリが強制終了
- 動作が重い

## 4. ユーザーが求めるもの

### 4.1 アプリ選びの重要ポイント

| 優先度 | 要素 |
|--------|------|
| 高 | 使いやすさ・シンプルさ |
| 高 | 直感的な操作性 |
| 中 | 達成感の可視化 |
| 中 | 適度なリマインド |
| 低 | 多機能性 |

### 4.2 目標設定のベストプラクティス

**SMART理論の活用:**
- Specific（具体的）
- Measurable（測定可能）
- Achievable（達成可能）
- Relevant（関連性のある）
- Time-bound（期限付き）

**その他のコツ:**
- 結果よりプロセスを目標にする
- 1日できなくても失敗ではないというマインド
- 高すぎる目標は途中で修正してOK

## 5. ターゲットセグメント

### 5.1 プライマリターゲット

**「シンプル志向の目標管理初心者」**

- 属性: 20-40代、個人利用
- 行動: 新年に抱負を立てるが、毎年挫折
- ペイン:
  - 既存アプリが複雑すぎる
  - 何から始めていいか分からない
  - 達成感を感じにくい
- ニーズ:
  - シンプルで直感的なUI
  - 視覚的な達成感
  - 年間を通じた目標の可視化

### 5.2 セカンダリターゲット

**「9x9マンダラは複雑すぎると感じる層」**

- 属性: マンダラチャートを知っているが使いこなせない
- ペイン: 81マスは多すぎて挫折
- ニーズ: より小さい単位（9マスや25マス）で始めたい

## 6. インサイト

### 6.1 重要な発見

1. **92%が挫折する市場** = 改善余地が大きい
2. **5月以降に意識が薄れる** = 定期的なリマインドが有効
3. **曖昧な目標が諸悪の根源** = 具体化を促すUIが必要
4. **広告への強い不満** = 広告少なめ or 買い切りモデルが差別化に

### 6.2 bingo_goal_appへの示唆

| ユーザーペイン | bingo_goal_appでの対応案 |
|----------------|-------------------------|
| 目標を忘れる | ビンゴ形式で常に目標を一望 |
| 達成感がない | ビンゴが揃う演出で達成感を強調 |
| 複雑すぎる | 3x3（9マス）からスタート |
| 年間で意識薄れる | 年度別管理で1年間の進捗を可視化 |
| 広告が邪魔 | 広告控えめ or 買い切りモデル |
